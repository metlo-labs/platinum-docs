---
title: "Synthesize a conversation"
api: "POST https://app.fluxon.dev/api/v1/tts/synthesize-sequence"
description: "This endpoint synthesizes is used to synthesize a conversation between multiple voices"
---

### Body

<ParamField body="voiceUuid" type="uuid">
  The id of the voice to use for the synthesis
</ParamField>

<ParamField body="text" type="uuid">
  The text to synthesize
</ParamField>

<ParamField body="pitch" type="int">
  The pitch of the voice to be synthesized
</ParamField>

<ParamField body="speed" type="int">
  The speed of the voice to be synthesized
</ParamField>

<ParamField body="delayAfter" type="float">
  The gap after a synthesized voice. Is optional.
</ParamField>

<ParamField body="delayBefore" type="float">
  The gap before a synthesized voice. Is optional.
</ParamField>

### Header

<ParamField header="Authorization" type="string">
  This is the user auth token generated for app.fluxon.dev.
</ParamField>

### Response

<ResponseField name="uuid" type="uuid">
  The id of the task synthesizing the conversation. This can be fetched by using the API at [GET Synthesize Sequence](/api-reference/tts/get_synthesize_sequence)
</ResponseField>
