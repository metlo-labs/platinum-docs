---
title: "List Available Voices"
api: "GET https://app.fluxon.dev/api/v1/tts/voices"
description: "This endpoint lists the voices available for speech synthesis"
---

### Header

<ParamField header="Authorization" type="string">
  This is the user auth token generated for app.fluxon.dev.
</ParamField>

### Response

<ResponseField name="id" type="uuid">
  The uuid of the voice
</ResponseField>

<ResponseField name="gender" type="string">
  The gender of the voice. May be *male*, *female*, or *other*.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the voice.
</ResponseField>

<ResponseField name="language" type="string">
  The languages supported by the voice.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://app.fluxon.dev/api/v1/tts/voices' \
--header 'Authorization: metlo.abcdEFGH12345678zyxw'
```

</RequestExample>
